The ledstrip class. 

This is the base class for the ledstrip library. This class defines all the base charachteristics of a ledstrip. All the \hyperlink{class_l_e_dstrip}{L\+E\+Dstrip} functions are declared and defined in this class.

Sets all the led. 

This functions sends the array with all the info to the strip.

The array with all the led info wil loop through the function. Depending if there is a 1 or 0 the Send\+Byte en Send\+Bit function will be called. Nothing will happen aslong you do not call the show function. At first the sendstart function will be called. After the start de array will be call. At the end the endframe will be called.
